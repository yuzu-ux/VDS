# Editorial Serif

Print-magazine confidence: generous whitespace, oversized serif headlines,
restrained palette of off-white paper + ink + a single warm accent. Think
Monocle, FT Weekend, It's Nice That.

## Tokens

| Token | Value | Use |
|---|---|---|
| `--bg` | `#f7f4ec` | paper |
| `--surface` | `#fffdf8` | cards |
| `--ink` | `#191817` | text |
| `--muted` | `#6b665c` | secondary |
| `--line` | `#ddd6c8` | rules |
| `--accent` | `#a33327` | warm red, used like a printer's second ink |

## Typography

- Display: serif (`"Iowan Old Style", Charter, Georgia`), large and unafraid — clamp up to 88px, weight 500, tight leading (1.05).
- Body: system sans at 16–17px, line-height 1.65.
- Mono for section numbers, captions, and datelines, letterspaced uppercase.
- Drop caps and pull-quotes are encouraged in long-form sections.

## Rules

1. Whitespace is the luxury — sections breathe at 96–128px vertical padding.
2. Horizontal rules (1px `--line`) structure the page like a broadsheet.
3. The red accent: eyebrows, drop caps, one CTA. Never backgrounds.
4. Full-bleed "photo" placeholder dividers between dense sections; honest placeholders with mono captions, never fake stock imagery.
5. Justified rag is forbidden; left-align everything except grand centered covers.
